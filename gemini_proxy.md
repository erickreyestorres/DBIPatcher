# Gemini Proxy API Reference

Для використання проксі із зовнішніх програм (наприклад, через `curl`, `Postman` або бібліотеку `requests`) підтримується як стандартний OpenAI-сумісний синтаксис, так і спеціальні розширені функції для керування браузером.

### 1. Створення нового чату (New Chat)
Для того, щоб скинути поточну сесію та почати "з чистого листа" в інтерфейсі AI Studio:
- **URL**: `http://127.0.0.1:2048/api/new-chat`
- **Метод**: `POST`
- **Тіло**: (порожнє)

### 2. Передача повідомлення (Chat Completion)
Використовується стандартний формат OpenAI.
- **URL**: `http://127.0.0.1:2048/v1/chat/completions`
- **Метод**: `POST`
- **Приклад тіла запиту**:
```json
{
  "model": "gemini-2.0-flash",
  "messages": [
    {"role": "user", "content": "Hello, how are you?"}
  ],
  "stream": false
}
```

### 3. Функції прокручування (Totop & Tobotom)
Ці функції використовуються для керування сторінкою в браузері:
- **Totop (Scroll to Top)**:
    - **URL**: `http://127.0.0.1:2048/api/scroll-to-top`
    - **Метод**: `POST`
- **Tobotom (Scroll to Bottom)**:
    - **URL**: `http://127.0.0.1:2048/api/scroll-to-bottom`
    - **Метод**: `POST`

### 4. Керування кнопками Run та Stop
Імітація натискання кнопок в інтерфейсі AI Studio:
- **Click Run (Запуск генерації)**:
    - **URL**: `http://127.0.0.1:2048/api/click-run`
    - **Метод**: `POST`
    - **Параметри (опціонально)**: `{"delay_ms": 500}`
- **Click Stop (Зупинка генерації)**:
    - **URL**: `http://127.0.0.1:2048/api/click-stop`
    - **Метод**: `POST`

### 5. Інші функції
- **Перегляд моделей**: `GET http://127.0.0.1:2048/v1/models`
- **Статус сервера**: `GET http://127.0.0.1:2048/api/server/status`
- **Health Check**: `GET http://127.0.0.1:2048/health`

---
*Примітка: Ці функції згруповані під префіксом `/api`, тоді як основна робота з моделями йде через `/v1`.*
