### 📋 Pipeline Disponible

```powershell
# Comando                  # Descripción
python -m src.main sync       # CSV → dictionary.xlsx
python -m src.main translate  # AI translation (necesita OmniRoad)
python -m src.main validate   # Multi-rule validation
python -m src.main align      # Column alignment
python -m src.main export     # dictionary.xlsx → CSVs
python -m src.main build      # CSVs → .bin files  
python -m src.main dist       # Organize for distribution
python -m src.main deploy     # Git + GitHub Release
python -m src.main check      # Health check
python -m src.main test       # Full pipeline (sync→dist)
python -m src.main all        # Full pipeline (sync→deploy)
python -m src.main help       # Show help
```
